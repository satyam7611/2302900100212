# Stage 1: Campus Notification System - Design Document

I created this document to explain the approach I followed while building the Campus Notifications application. The goal was to help students quickly identify important notifications related to placements, results, and events without getting overwhelmed by a large number of updates.

While implementing the solution, I focused on three main areas:

* Showing the most important unread notifications first.
* Keeping the UI responsive and easy to use.
* Making sure new notifications can be handled efficiently as they arrive.
